🚀 QUICK START (3 COMMANDS)
═══════════════════════════════════════════════════════════════════

1. Install dependencies:
   pip install -r gui/gui_requirements.txt

2. Start API server (Terminal 1):
   python3 -m uvicorn app.main:app --reload

3. Start GUI application (Terminal 2):
   python3 gui/main.py
